HexoとGithub pagesでブログ環境を作ってみよう

ブログ環境を用意するにあたり、はてなとかTumblrとか色々見たあげく、github pagesを使えばGoogle Adsenseも使えることがわかり、勉強もかねてHexoという静的サイトジェネレータで作ってみました。

Hexoとは

HexoとはStatic Site Generatorsでも2017年7月9日現在でJekyll、Hugoに続いて第３位と勢いのあるnode.js製の静的サイトジェネレーターです。
記事はMarkDown方式で書くことができ、カンタンなコマンドでページの追加をしたり、テーマの利用ができます。

Hexo導入まで

node.jsが入っていることが前提になるので、もし入っていない場合はインストールしましょう。
(私はligさんのページを参考にしました。)
また、事前にgithub pagesのためのリポジトリの準備が必要になります。(公式ページ)

ほとんどはligさんのページの通りでいいのですが、私が詰まったポイントを以下に記載しておきます。

テーマの適用方法

Hexoで使えるテーマはこちらで探せますが、「Hexo テーマ おすすめ」とかでググれば、まとめも出てくるので、そちらを参考にするのがいいかもしれません。
ちなみに、このページは”apollo”というテーマを使っています。
テーマの当て方はこちらを参考にしましたが、基本的には、

  • gitからcloneしてきて、themesの下に格納
  • config.ymlのthemeの設定を変更するだけでOK。

config.ymlの設定を変えるときにはthemesに入れたフォルダ名を入れます。

themeをlandscapeからapolloに変更。念のためコメントアウトしているだけなので、削除してもOK。
